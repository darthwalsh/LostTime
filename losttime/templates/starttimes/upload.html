The csv format follows the format displayed in the WIOL registration administration page: first column SCHOOL, followed by columns for each course. Extra columns are ignored. Blanks for any column are interpreted as zero. No transformation of data should be required.

CSV Column Header Definition
SCHOOL Full School Name
ELEM Number of starts on the Elementary course to allocate for this school.
MID Number of starts on the Middle School course to allocate for this school.
JVG Number of starts on the JV Girls course to allocate for this school.
JVBN Number of starts on the JV Boys North course to allocate for this school.
JVBS Number of starts on the JV Boys South course to allocate for this school.
VAR Number of starts on the Varsity course to allocate for this school.
